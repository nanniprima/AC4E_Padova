# CLAUDE.md For Economics Research

## Project

This is an economics research project. Before editing, identify the research
question, allowed files, forbidden files, and verification step.

## Working Agreements

- Start with a short plan for multi-file edits.
- Protect `data/raw/` and confidential manuscripts.
- Do not invent citations, results, or data sources.
- Prefer `statsmodels` or discipline-appropriate econometric tools for inference.
- Report commands run and verification evidence.
- Use a referee-style checklist for paper review tasks.

## Review Standard

For paper or code review, lead with decision-relevant issues. Separate major
issues from optional improvements. Every major issue must cite a concrete
location in the draft, code, table, or equation.
