# Claude Code App Track (Lane B2)

Same `.claude/` harness as CLI. Desktop uses parallel sessions and in-app `/` menu.

## Setup

1. Code tab → project folder = your `starter_article` copy.
2. Merge files per [`../claude/README.md`](../claude/README.md).
3. Invoke skills from `/` menu or prompt by name.

## Surface notes

- Desktop thread history does **not** resume via `claude --continue` from CLI.
- Use checkpoints inside the app session for long tasks.

See [`docs/research_article_harness.md`](../../docs/research_article_harness.md).
