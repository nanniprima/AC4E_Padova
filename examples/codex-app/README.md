# Codex App Track (Lane A2)

Same project files as **Codex CLI** (`.codex/skills`, hooks). Differences are UI-only.

## Setup

1. Open Codex desktop app → add project folder (your `starter_article` copy).
2. Merge harness from [`../codex/README.md`](../codex/README.md).
3. Trust hooks via `/hooks` in the thread (not only terminal).

- Slash commands match CLI: `/init`, `/review`, `/goal` where enabled.
- Session resume uses app threads, not `codex resume` from a different terminal session.
